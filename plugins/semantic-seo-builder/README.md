# Semantic SEO Builder

A full-pipeline Cowork plugin for building local business websites using Koray Tuğberk Gübür's Semantic SEO framework — from market research to live rankings.

## What this plugin does

This plugin encodes the complete methodology behind real-world results:
- 15–20 form submissions daily
- 20+ calls per day
- #1 rankings for primary queries
- Local 3-Pack placement
- AI citation (entity trust signal)

## Skills

| Skill | Command | What it produces |
|-------|---------|-----------------|
| Full pipeline | `/workflow` | Runs all 5 phases sequentially, files at each stage |
| Foundation & Strategy | `/foundation` | Market research doc, ICP profiles, intent map |
| Semantic Research | `/semantic-research` | Entity map, attribute matrix, relationship graph |
| Topical Map | `/topical-map` | Raw + processed topical map, pillar structure |
| Content Brief | `/content-brief` | Semantic brief for any page or article |
| Technical & Local SEO | `/technical-local` | Site architecture plan, GBP alignment checklist |

## The 5-Phase Framework

```
Phase 1 → Foundation & Strategy
Phase 2 → Semantic Research
Phase 3 → Topical Authority Building
Phase 4 → Content System
Phase 5 → Technical & Local SEO Layer
```

## How to use

Start with `/workflow` for a guided end-to-end build. Or jump into any individual phase skill if you need a specific deliverable.
