# Semantic SEO Plugin Marketplace

Claude Code plugin marketplace by [suvtrade](https://github.com/suvtrade).

## Install

```
/plugin marketplace add suvtrade/semantic-seo-plugins
/plugin install semantic-seo-builder@suvtrade-seo
```

## Plugins

### semantic-seo-builder v0.8.0

Koray Tugberk Gubur Semantic SEO framework for local businesses.

**Track 1 — New site build (5 phases):**
- Phase 1: Foundation — customer voice research, brand persona, competitor baseline
- Phase 2: Semantic Research — entity/attribute matrices, SERP cluster validation
- Phase 3: Topical Map — hub-and-spoke architecture with intent validation
- Phase 4: Content System — brief → write → factcheck → schema → GEO → images
- Phase 5: Technical & Local SEO — full audit, schema (with floorLevel), local signals

**Track 2 — Ongoing operations:**
- Weekly pipeline: write → factcheck → seo-check → schema → GEO → images → publish → GBP post → drift check
- Monthly audit: rankings, content refresh, AI citation tracking, backlink health

**Integrates:** claude-seo and claude-blog plugins for best results.
